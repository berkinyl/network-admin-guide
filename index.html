Final Design Project Overview

Welcome to your Network course Final Design Project (FDP). You will identify ten current network issues spanning OSI Layers 7 down to 1, describe each problem clearly, and provide step-by-step solutions. Assemble everything as an Admin Guide Tutorial in a navigable HTML format.

Include Cisco Packet Tracer simulations for each scenario, and package the CPT files alongside your HTML. A bonus will be awarded for an interactive homepage linking to each issue page with embedded or downloadable simulations.

Deadline: One week after the final exam. FDP counts for 60% of your final grade and is only accepted if you sat the final exam.

Modules by OSI Layer

[L1] Ethernet Cabling Standard Violation and Data Corruption
[L2] Unauthorized Access Due to Lack of Port Security
[L2] VLAN Misconfiguration Causing Broadcast Leakage
[L3] Unauthorized Network Access Due to Missing ACLs
[L3] OSPF Router ID Conflicts Causing Route Instability
[L4] Media Disruption Due to Lack of QoS on UDP Traffic
[L5] Session Persistence Issues After VPN Disconnection
[L6] Data Conflicts Due to Encoding Format Mismatch
[L7] Traffic Redirection Through DNS Spoofing
[L7] Security Risks of Using HTTP Instead of HTTPS
